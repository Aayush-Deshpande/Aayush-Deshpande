## ▸ Techniques Deployed

<table>
<tr>
<td width="50%" valign="top">

### 🌕 PRISM
*The flagship.*

Lunar rover RL simulation built on real **Chandrayaan-2** DFSAR + OHRC data. Ice detection, a physics-accurate terrain digital twin, and a PPO agent navigating it with Dijkstra/A* warm starts.

</td>
<td width="50%" valign="top">

### 🧠 FedNeMo
Federated fine-tuning framework on **Nemotron**, with a custom FedRand split-adapter mechanism, adaptive 2-bit quantization, and entropy-aware aggregation.

</td>
</tr>
<tr>
<td width="50%" valign="top">

### ⚖️ Leon
Legal RAG agent with hybrid dense + BM25 retrieval, where self-critique lives inside the agent's own decision loop instead of being bolted on afterward.

</td>
<td width="50%" valign="top">

### ⚙️ AutoML
Feed it any dataset, it does the rest — preprocessing, feature selection, multi-model training, metric comparison, hyperparameter tuning. Walk away, come back to a leaderboard.

</td>
</tr>
</table>
