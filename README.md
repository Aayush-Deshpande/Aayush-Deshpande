### From research to shipped systems

<img src="https://raw.githubusercontent.com/Aayush-Deshpande/Aayush-Deshpande/main/assets/systems_map.svg" width="100%" alt="systems map"/>

<br>

<details open>
<summary><b>PRISM — lunar rover RL simulation</b></summary>
<br>

Built on real **Chandrayaan-2** DFSAR + OHRC data. Subsurface ice detection via CPR/DOP scoring, a physics-accurate terrain digital twin with solar irradiance modeling and battery physics, Dijkstra/A* warm-start pathfinding, and a PPO agent (stable-baselines3) handling adaptive navigation. Outputs landing site proposals end to end.

</details>

<details>
<summary><b>FedNeMo — federated fine-tuning on Nemotron</b></summary>
<br>

Custom **FedRand** mechanism — split-adapter approach with differential privacy — plus adaptive 2-bit quantization and entropy-aware aggregation. A second Nemotron instance runs at inference time as a trust-scoring agent over incoming updates.

</details>

<details>
<summary><b>Leon — legal-domain RAG agent</b></summary>
<br>

Hybrid dense + BM25 retrieval. Self-critique is embedded directly in the agent's decision loop, gating each next action on retrieval relevance and citation-groundedness — not a bolt-on evaluation step.

</details>

<details>
<summary><b>AutoML — end-to-end ML pipeline</b></summary>
<br>

FastAPI backend, Streamlit frontend. Feed it a dataset — preprocessing, feature selection, multi-model training, hyperparameter tuning, leaderboard. Walk away, come back to results.

</details>
